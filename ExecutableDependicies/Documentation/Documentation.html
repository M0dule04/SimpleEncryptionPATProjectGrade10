Derich Havenga IT PAT 2020 Documentation

About:

The software takes a text message and converts it to gibberish with a unique code and only with the unique code or key can it be translated.

How to use software:

  • Step one: Press Browse or Enter a message into the text field.
  • Step two: Select either Encrypt or decrypt.
  • Step three: Enter a ten digit code ex. ABCDEFGHIJ
  • Step four:Press the encrypt / decrypt button.
  • Step five: Sit back and relax for a few milliseconds (Optional).
  • Step six: Save in text document or (copy) text.

Troubleshooting

  • When I decrypt the scentence isn't readable. solution: You didn't enter the correct keycode.
  • The programm does not allow me to press the encrypt/decrypt button Solution: Enter a keycode or enter a keycode that is 10 characters in length.
